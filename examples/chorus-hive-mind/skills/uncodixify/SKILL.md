# Uncodixify

Read and apply [`RULES.md`](RULES.md) before writing UI code.

This local adaptation is derived from [cyxzdev/Uncodixfy](https://github.com/cyxzdev/Uncodixfy), which is MIT licensed. It was tightened around composition, brand presence, hero discipline, card reduction, typography, color, motion, and visual clutter.

When generic frontend defaults conflict with these rules, prefer the rules.
